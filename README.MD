🛒 E-commerce Data Analysis Using SQL



📌 Project Overview



This project analyzes an E-commerce dataset using SQL to extract business insights related to customer behavior, revenue performance, product trends, and sales funnel efficiency.



The goal of this project is to demonstrate practical SQL skills used in real-world data analytics workflows.



\---



🎯 Objectives



The analysis focuses on:



\- Understanding revenue distribution across products

\- Measuring customer conversion performance

\- Identifying refund patterns

\- Detecting drop-offs in the purchase funnel

\- Finding top-performing customers and products

\- Analyzing landing page effectiveness



\---



🛠 Tools Used



\- PGSQL 

\- GitHub

\- Data Cleaning Techniques

\- Business KPI Analysis



\---



📂 Dataset Description



The dataset contains E-commerce transactional data including:



\- Orders

\- Order Items

\- Customers

\- Products

\- Website Sessions

\- Refund Information



These tables were used to generate business insights through structured SQL queries.



\---



📊 Key Business Questions Answered



This project answers the following important business questions:



1\. What is the total revenue generated?

2\. Which products generate the highest revenue?

3\. What is the monthly sales trend?

4\. What is the refund rate by product?

5\. Which customers contribute the most revenue?

6\. What is the average order value?

7\. Where do users drop off in the purchase funnel?

8\. Which landing page has the highest conversion rate?
